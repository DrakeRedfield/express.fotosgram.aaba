# Backend Fotosgram

Backend para la aplicación de (Ionic FotosGram)[https://github.com/DrakeRedfield/ionic.fotosgram.aaba] hecho con Express.

# Installación

Instalar todos los plugins necesarios para poder ejecutar Express

```
npm install
```
